Dream Music

powered by JavaFX

A music player that has almost the basic features of a music player
Written by Amir Ali Azimi

Features :

  • music volume
  • music loop
  • forward
  • rewind
  • music rate
  • random player
  • music downloader
  • system dark mode
  • mini player

this app is also open source you can go to github page and see the source code
